# NeoVim commands

### Neotree

<C-n> = open file tree

### Telescope

<leader>ff = find files
<leader>fg = live grep
<leader>fb = buffer
<leader>fh = help tags

### LSP

shift+K = function/code information
<leader>gd = definition
<leader>gr = reference
<leader>ca = code action

### NoneLS aka NullLS

<leader>gf = format file

### Completions

<C-b> = scroll docs -4
<C-f> = scroll docs 4
<C-Space> = complete
<C-e> = abort
<CR> = confirm

### Debugger

<leader>dt = toggle breakpoint
<leader>dc = continue
